# intera
Learning Platform

# requirements
The platform shall contain a flashcard tool
  The flashcard tool shall allow the user to create flashcards
    The flashcard tool shall allow the user to modify existing flashcards
    The flashcard tool shall allow the user to delete flashcards
  The flashcard tool shall allow the user to create decks of flashcards
  The flashcard tool shall allow the user to reveal the answer of a flashcard
    The flashcard tool shall allow the user to answer the flashcard using voice
    The flashcard tool shall allow the user to answer the flashcard using text
  The flashcard tool shall allow the user to mark the card as correct or incorrect
    The flashcard tool shall mark the card as correct or incorrect based on user response
    The flashcard tool shall provide an optional confidence rating in the answer from 1 to 10
    The flashcard tool shall generate a confidence rating based on time of response and quality of response
  The flashcard tool shall utilize a spaced repetion learning algorythm to maximize learning rate and retention of flashcards
    https://github.com/open-spaced-repetition/free-spaced-repetition-scheduler
    https://github.com/vlopezferrando/simple-spaced-repetition/blob/main/simple_spaced_repetition.py
  The flashcard tool shall send push notifications to the user to remind them to study
  The flashcard tool shall create a learning plan based on deadlines for knowing the content
